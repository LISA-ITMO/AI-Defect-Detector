### Installation

---
#### Clone this repository
`
git clone git@github.com:pol3et/defect-detector.git
`

#### Install dependencies
`
pip install requirements.txt
`

#### cd to the project directory
`
cd defect-detector
`

#### Launch the app
`
python gui.py
`
